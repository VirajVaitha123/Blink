# Alembic migrations

Run from `backend/` once SQLModel tables exist in `blink_shared/models/tables/`:

```bash
cd backend
uv add alembic --package blink_shared
uv run alembic init alembic
```

Then update `alembic/env.py` to import models and use `SQLModel.metadata`:

```python
import sys
from pathlib import Path

sys.path.insert(0, str(Path(__file__).parent.parent))

from sqlmodel import SQLModel
from blink_shared.config import settings
from blink_shared.database import engine
from blink_shared.models.tables import *  # noqa: F401,F403 - register all models

config.set_main_option("sqlalchemy.url", str(settings.database_url))
target_metadata = SQLModel.metadata
```

Generate / apply migrations:

```bash
uv run alembic revision --autogenerate -m "Initial tables"
uv run alembic upgrade head
```

If autogenerated migrations contain `sqlmodel.sql.sqltypes.AutoString()`, replace with `sa.String()`.
